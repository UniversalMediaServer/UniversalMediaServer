Plugins

About plugins

UMS can be extended with plugins. A plugin file can be installed by copying it into the "plugins" directory. When UMS is started, all installed plugins are read from the directory. Some plugins have options that can be edited. Click on the button for the plugin in the "General Configuration" tab, under "Plugins".
