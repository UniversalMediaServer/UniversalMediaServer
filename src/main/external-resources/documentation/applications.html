External applications

IMPORTANT: This help page is not more maintained.

See our online maintained documentation instead.

UMS supports the use of external applications to transcode media. These applications are:

  • VLC media player
  • AviSynth
  • CoreAVC

For the details on the exact working of each application, see its manual.
